Реализован telegram бот с авторизацией и простое приложение калькулятор
Имя бота в telegram: t.me/QA_Automation_Testing_SOng_bot
Реализована "авторизация" по логину и паролю
	После запуска бота(/start) следует отпрвить сообщение с логином и паролем
БД используется Mongodb(локально) c несколькими тестовыми пользователями, например: firstUser qwerty123

Примечание:
	так как калькулятор должен быть простым то в нем не реализована правильная приоритезация операций, например:
		для выражения 6/2*4-7*5 будет получен результат -34.25 хотя корректный −23
	в боте t.me/QA_Automation_Testing_bot - поведение аналогичное
